# Sprint 1 Plan

## Sprint Goal

Build the core chat system where users can connect with a random partner and exchange messages in real time.

---

## Selected User Stories

1. As a user, I want to enter a chat queue, so that I can be matched with a random person. (Story Points: 5)

2. As a user, I want to send messages, so that I can communicate with my chat partner. (Story Points: 5)

3. As a user, I want to receive messages, so that I can read what my partner sends. (Story Points: 3)

4. As a user, I want to leave the chat, so that I can exit anytime. (Story Points: 2)

5. As a user, I want to start a new chat after ending one, so that I can continue chatting with others. (Story Points: 3)

---

Total Story Points: 18

---

## Task Assignment

Member 1 – Enter chat queue
Member 2 – Send messages
Member 3 – Receive messages
Member 4 – Leave chat and testing
Member 5 – Start new chat, documentation, and Git management
