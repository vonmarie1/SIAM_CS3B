User Story 1

As a user, I want to enter a chat queue, so that I can be matched with a random person.

Priority: High
Story Points: 5

Acceptance Criteria:

 User can click a “Start Chat” button
 System adds user to queue
 System searches for another available user
 User is notified when matched

User Story 3

As a user, I want to receive messages, so that I can read what my partner sends.

Priority: High
Story Points: 3

Acceptance Criteria:

 Incoming messages appear instantly
 Messages are displayed in correct order
 Messages are readable and formatted
User Story 4

As a user, I want to leave the chat, so that I can exit anytime.

Priority: High
Story Points: 2

Acceptance Criteria:

 User can click “Leave Chat”
 System disconnects user
 Chat session ends immediately
 User returns to home screen
