# Project Schedule (15 Weeks)

| Week    | Goals                                   | Deliverables                                            |
| ------- | --------------------------------------- | ------------------------------------------------------- |
| Week 1  | Project setup and planning              | GitHub repo created, initial README, team setup         |
| Week 2  | Requirements gathering and user stories | `backlog.md` with 10+ user stories                      |
| Week 3  | Sprint planning and team organization   | `sprint-1-plan.md`, `team-roles.md`                     |
| Week 4  | System design and architecture          | Basic system flow diagram, database/chat structure plan |
| Week 5  | Development setup                       | Working development environment, dependencies installed |
| Week 6  | Core backend setup                      | Chat server, basic connection handling                  |
| Week 7  | Chat queue system                       | Users can enter queue and get matched                   |
| Week 8  | Messaging system (real-time)            | Send and receive messages between users                 |
| Week 9  | Session management                      | Handle disconnects and chat termination                 |
| Week 10 | UI development                          | Basic chat interface (frontend)                         |
| Week 11 | New chat and leave features             | Users can leave and restart chats                       |
| Week 12 | Integration testing                     | Backend + frontend fully connected                      |
| Week 13 | Bug fixing and improvements             | Stability improvements and fixes                        |
| Week 14 | Final testing and optimization          | Performance testing, UI polish                          |
| Week 15 | Final documentation and submission      | Final report, clean code, deployment-ready project      |
